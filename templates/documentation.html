Bienvenue sur Scientral

Votre laboratoire scolaire, enfin organisé

Scientral est une application SaaS de gestion de laboratoire scolaire (SVT / Physique-Chimie) conçue pour simplifier le quotidien des enseignants et des gestionnaires. Toutes vos données sont centralisées, accessibles en un clic, et sécurisées dans le cloud.

Centralisation

Tout votre matériel, vos produits et vos documents au même endroit.

Planification

Un calendrier intuitif pour éviter les conflits de réservation entre collègues.

Sécurité

Suivi rigoureux des maintenances réglementaires et des produits dangereux.

Bon à savoir : Scientral fonctionne sur ordinateur, tablette et smartphone. Aucune installation n'est requise — il suffit d'un navigateur web.

Gestion de l'Inventaire

Organisez, suivez et gérez tout votre matériel

L'inventaire est le cœur de Scientral. Il est structuré en deux niveaux : les Armoires (localisation physique) et les Catégories (type de matériel).

1
Créez vos armoires et catégories

Via le menu "Gérer les Armoires" et "Gérer les Catégories". Vous pouvez en créer autant que nécessaire et y associer une photo.

2
Ajoutez vos objets

Depuis la page "Inventaire Complet", cliquez sur + Ajouter un matériel. Renseignez le nom, la quantité, le seuil d'alerte, l'armoire et la catégorie.

3
Suivez les stocks en temps réel

Le tableau de bord affiche automatiquement les alertes de stock bas et les produits à péremption proche.

L'inventaire est structuré par Armoires (localisation physique) et par Catégories (type de matériel). Vous pouvez créer autant d'armoires et de catégories que nécessaire via le menu "Gérer".

Chaque armoire peut avoir une photo pour faciliter l'identification visuelle. Cliquez sur une armoire pour voir tous les objets qu'elle contient.

Pour les produits chimiques, vous pouvez associer une FDS (PDF ou lien URL). Une icône apparaîtra dans la liste pour un accès rapide en cas d'urgence.

Les FDS sont consultables directement depuis la fiche de l'objet, sans avoir à chercher dans des dossiers papier.

Les kits permettent de regrouper plusieurs objets sous un même nom (ex: "Kit dissection CM2"). Lors d'une réservation, réservez le kit en une seule action plutôt que chaque objet individuellement.

Les kits sont créés et gérés par l'administrateur depuis le panneau d'administration.

La barre de recherche globale (en haut de chaque page) permet de trouver n'importe quel objet par son nom ou son code-barre.

Sur la page Inventaire, des filtres supplémentaires permettent de trier par armoire, catégorie ou état (stock bas, périmé...).

Système de Réservation

Planifiez sans conflits, collaborez efficacement

Le module de réservation empêche les conflits de stock en temps réel. Lorsque vous réservez un objet pour une date donnée, il est automatiquement indisponible pour vos collègues sur cette période.

Le Panier

Ajoutez des objets ou des kits à votre panier. Le système vérifie instantanément la disponibilité pour la date choisie et vous alerte en cas de conflit.

Le Calendrier

Visualisez toutes les réservations sur une vue mensuelle ou journalière. Cliquez sur un jour pour voir le détail de toutes les réservations.

1
Depuis l'inventaire, cliquez sur "Réserver"

Sur la fiche d'un objet, le bouton "Ajouter au panier" vous permet de le réserver pour une date précise.

2
Choisissez vos créneaux

Définissez la date, l'heure de début et de fin, et la salle si nécessaire. Le système vérifie la disponibilité en temps réel.

3
Validez votre réservation

Une confirmation apparaît et votre réservation est visible sur le calendrier partagé. Vous la retrouverez aussi dans "Mes Réservations" sur le tableau de bord.

Astuce : Utilisez les Kits pour réserver en un clic tout le matériel nécessaire à un TP, plutôt que d'ajouter chaque objet un par un.

Sécurité & Maintenance

Conformité réglementaire et suivi des équipements

Ce module critique permet de suivre la conformité des équipements (Hottes, Extincteurs, Douches de sécurité, Trousses de secours...) et d'enregistrer les interventions de maintenance.

Suivi des contrôles

Enregistrez chaque vérification avec date, résultat et intervenant.

Alertes automatiques

Recevez une alerte dès qu'un équipement approche de sa date de contrôle.

Signalement incident

Tout utilisateur peut signaler un dysfonctionnement en temps réel.

Signalement d'incident

Tout utilisateur peut signaler un dysfonctionnement via le bouton "Signaler" sur la fiche d'un équipement. L'administrateur reçoit une alerte immédiate sur son tableau de bord et peut traiter la demande depuis le panneau d'administration.

Suivi Budgétaire

Gérez les dépenses de votre laboratoire par année scolaire

Le module budgétaire permet à l'administrateur de définir un budget annuel et d'enregistrer toutes les dépenses au fil de l'année scolaire.

Fonctionnalité Description Accès
Années scolaires Budget géré par année. Basculez via le sélecteur. Admin
Dépenses Enregistrez vos commandes. Solde mis à jour en temps réel. Admin
Consultation Tous les utilisateurs peuvent consulter le budget. Tous
Export Exportez les dépenses en CSV ou Excel. Tous
Clôture Fige les comptes en fin d'année. Action irréversible. Admin
Clôture annuelle : En fin d'année scolaire, l'administrateur peut clôturer le budget pour figer les comptes. Cette action est irréversible — assurez-vous que toutes les dépenses sont bien enregistrées avant de procéder.

Alertes & Suggestions

Restez informé, anticipez les manques

Scientral surveille en permanence vos stocks et vous alerte automatiquement dès qu'une action est nécessaire.

Stock bas

Lorsque la quantité d'un objet passe sous son seuil d'alerte, une notification apparaît sur l'icône de cloche dans le header.

Péremption proche

Les produits dont la date de péremption approche (moins de 30 jours) sont signalés en orange dans l'inventaire.

Suggestions de réapprovisionnement

Les utilisateurs peuvent suggérer un réapprovisionnement depuis la fiche d'un objet. L'admin traite les demandes depuis le tableau de bord.

Échéances réglementaires

Les prochaines échéances (vérifications, renouvellements...) sont affichées sur le tableau de bord avec un compte à rebours.

Support & Dépannage

Nous sommes là pour vous aider

La licence est liée à l'Identifiant unique de votre installation (Instance ID). Si vous changez d'ordinateur ou réinstallez l'application, cet identifiant change et la licence se désactive par sécurité.

Solution : Contactez le support en fournissant votre nouvel "Identifiant Instance" (visible dans la page À Propos) pour obtenir une nouvelle clé d'activation.

Si une page affiche une erreur ou ne fonctionne pas comme prévu :
  1. Essayez de rafraîchir la page (F5).
  2. Vérifiez que vous êtes bien connecté.
  3. Videz le cache de votre navigateur (Ctrl+Shift+Delete).
  4. Si le problème persiste, contactez le support en décrivant précisément l'action qui a provoqué l'erreur.

  • Vérifiez que vous utilisez la bonne adresse email.
  • Utilisez le lien "Mot de passe oublié" sur la page de connexion.
  • Si vous n'avez pas reçu l'email de réinitialisation, vérifiez vos spams.
  • Si votre compte a été désactivé, contactez votre administrateur.

Scientral est conçu pour fonctionner sur tous les appareils. Si l'affichage est incorrect :
  • Assurez-vous d'utiliser un navigateur récent (Chrome, Safari, Firefox).
  • Essayez de vider le cache de votre navigateur mobile.
  • Sur iPhone, ajoutez l'application à votre écran d'accueil pour une expérience optimale.

Administration

Outils réservés aux administrateurs

Cette section est réservée aux utilisateurs avec le rôle Administrateur.
Sauvegardes

Exportez toutes vos données (Inventaire, Historique, Paramètres) dans un fichier JSON sécurisé. C'est votre assurance vie : faites-le régulièrement.

Import Excel

Ajoutez des centaines d'objets en une fois grâce au modèle Excel fourni. Idéal pour l'initialisation du laboratoire.

Continuité de Service (Changement de PC)

Si vous devez changer d'ordinateur ou réinstaller l'application, suivez cette procédure pour ne rien perdre :

  1. Sur l'ancien poste, allez dans Admin › Sauvegardes et cliquez sur "Télécharger la base".
  2. Installez Scientral sur le nouveau poste (créez un compte admin temporaire).
  3. Allez dans Admin › Sauvegardes et importez votre fichier JSON.
  4. Tout est restauré : vos données, vos comptes utilisateurs et votre Licence PRO.

Architecture Technique

Pour les curieux et les intégrateurs

Cette section est destinée aux administrateurs techniques.
Backend
  • Python / Flask : Moteur applicatif léger et sécurisé.
  • SQLAlchemy : ORM pour la gestion des données relationnelles.
  • PostgreSQL : Base de données hébergée sur Neon (cloud).
  • Flask-Mail : Envoi des emails de réinitialisation.
Infrastructure & Services
  • Render : Hébergement de l'application (PaaS).
  • Neon : Base de données PostgreSQL serverless.
  • Cloudinary : Stockage et CDN des images et documents.
  • Bootstrap 5.3 : Framework CSS frontend.
